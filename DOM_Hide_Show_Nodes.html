Note: Simplified Example

This example has been simplified to highlight the functions for creating a show/hide effect using JavaScript. An actual implementation would also provide an indication of the current state of the show/hide control (i.e. is it hidden currently?). For example, instead of using two controls, one for hide and one for show, you might use one control that changes based on the current state. See PG 20 for information and examples on this requirement.
